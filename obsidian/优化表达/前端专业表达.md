
# 侧边栏（Sidebar）

## 侧边栏

在前端中，“侧边栏”通常指的是页面一侧的导航或辅助区域

## <font style="background-color:rgba(255, 255, 255, 0);">侧边栏中的功能选项(</font>**菜单项 **<font style="background-color:rgba(255, 255, 255, 0);">)</font>

+ **菜单项 (Menu Item)：** 这是最标准的叫法，指代导航列表中的每一个具体功能入口。

+ **导航项 (Navigation Item / Nav Item)：** 强调其作为跳转链接或页面切换的导航功能。

+ **列表单元 (List Item)：** 在代码实现层面（如 React 或 Vue 的组件库），这些选项通常被视为一个垂直列表中的单个单元。

# <font style="background-color:rgba(255, 255, 255, 0);">Favicon 右侧的文字提示（</font>**页面标题**<font style="background-color:rgba(255, 255, 255, 0);">）</font>

在浏览器标签页中，**Favicon 右侧的文字**通常称为： **页面标题（Title / Page Title）**

  

具体说明：

  

+ 这是 HTML 中 `<title>` 标签定义的内容

+ 显示在浏览器 Tab 上，位于 Favicon 右侧

+ 同时也会作为：

- 浏览器历史记录的标题

- 收藏夹（书签）的默认名称

  
  
  

# 文本输入框内的灰色文字（**占位符 placeholder**）

在前端设计中，输入框内的灰色提示文字通常称为 **占位符（placeholder）**。

  
  
  
  
  
  
  

更具体说明：

  

+ 它是在输入框为空时显示的提示文本

+ 用户开始输入后会自动消失

+ 常用于提示输入格式或示例（如“请输入手机号”）

  

在 HTML 中对应属性是：

  

+ `placeholder`

  
  
  
  
  

# 悬停触发组件

## 1. 提示类：Tooltip（文字提示）

> 吐提普（tǔ tí pǔ）

>

  

这是最基础的悬停组件。

  

+ **特点**：纯文字，内容简短，背景通常为深色。

+ **用途**：解释图标含义、按钮功能或截断文字的完整显示。

+ **交互约束**：不可点击，鼠标离开即消失。

# 三点菜单 / 三点图标

在前端设计中，这种 “…” 通常叫：

  

**1. 更多操作（More / More actions）**

最通用的命名，表示“这里还有额外功能”。

  

**2. 溢出菜单（Overflow menu）**

更偏设计/规范术语，意思是“放不下或不常用的操作被收纳到这里”。

  

**3. 三点菜单 / 三点图标（Ellipsis menu / Kebab menu）**

  

+ 水平三个点（…）：常叫 **Ellipsis menu**

+ 垂直三个点（⋮）：常叫 **Kebab menu**
